MaxBettiNumbers :: HilbertFunctions

HilbertFunctions

Description

Used as a key in MaxBetti with value being a VerticalList. A list of truncated Hilbert functions returned by maxBettiNumbers. See maxBettiNumbers(..., ResultsCount => ...) for more details.

Caveat

If Algorithm=>"Simplified" is forced, this may not return valid Hilbert functions for some inputs.

See also
