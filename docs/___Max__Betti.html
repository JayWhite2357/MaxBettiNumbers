MaxBettiNumbers :: MaxBetti

MaxBetti

Description

This is the type that is returned by maxBettiNumbers it is a HashTable with the following keys.

  • BettiUpperBound - upper bound for the total Betti numbers.
  • HilbertFunctions - a list of Hilbert functions with maximal total Betti numbers. See maxBettiNumbers(..., ResultsCount => ...) for more details.
  • isRealizable - if there is an ideal with the upper bound as its total Betti numbers.
  • MaximalBettiNumbers - the maximal total Betti numbers.
  • MaximumBettiSum - maximum sum of the total Betti numbers.

See also

  • maxBettiNumbers -- Upper bounds for total Betti numbers in a family of saturated ideals.
