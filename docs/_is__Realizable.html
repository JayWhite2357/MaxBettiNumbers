MaxBettiNumbers :: isRealizable

isRealizable

Description

Used as a key in MaxBetti with value being a Boolean. Is true if there is an ideal in the family that has total Betti numbers that match BettiUpperBound otherwise is false. See maxBettiNumbers(..., ResultsCount => ...) for more details.
