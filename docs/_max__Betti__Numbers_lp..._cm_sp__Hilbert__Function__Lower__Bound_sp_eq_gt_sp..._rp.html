MaxBettiNumbers :: maxBettiNumbers(..., HilbertFunctionLowerBound => ...)

maxBettiNumbers(..., HilbertFunctionLowerBound => ...)

Description

The options HilbertFunctionLowerBound, HilbertFunctionUpperBound, HilbertFunctionLowerBound, HilbertDifferenceLowerBound, and HilbertPolynomial are arguments to maxBettiNumbers. Each of these options, other than HilbertPolynomial, are a list of integers starting at degree 0. In the case where a value is not given, the corresponding constraint is removed (i.e. made the trivial constraint). Note: if HilbertPolynomial is not specified HilbertFunctionLowerBound and HilbertFunctionUpperBound must be equal for large degrees.

In the case where no lower bound is desired at a specified degree, 0, null, or nothing can be put instead. For instance, to specify only a lower bound of 4 in degree 3 on the Hilbert difference function, the option HilbertDifferenceLowerBound=>{,,,4} can be used.

Similarly, in the case where no upper bound is desired at a specified degree, infinity, null, or nothing can be put instead. For instance, to specify only a upper bound of 4 in degree 3 on the Hilbert function, the option HilbertFunctionUpperBound=>{,,,4} can be used.

There are some instances when these options, along with HilbertPolynomial conflict. In this case, an error is raised.

Further information
