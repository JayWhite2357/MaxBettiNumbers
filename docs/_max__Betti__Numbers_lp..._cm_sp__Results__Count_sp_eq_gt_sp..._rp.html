MaxBettiNumbers :: maxBettiNumbers(..., ResultsCount => ...)

maxBettiNumbers(..., ResultsCount => ...)

Description

The method maxBettiNumbers finds the upper bounds for the total Betti numbers. In certain instances, there are ideals that realize these upper bounds and have maximum possible total Betti numbers. In this case, these ideals also must have the maximum possible sum of the total Betti numbers. However, there are some instances where there are no ideals that realize the upper bounds. In this case, there are only ideals that realize maximal total Betti numbers. Some of these also must have the maximum possible sum of the total Betti numbers, while others do not.

ResultsCount is an option that can be passed to maxBettiNumbers. It determines how many, and what type of ideals are collected. The Hilbert function of these ideals is returned in a MaxBetti object under key HilbertFunctions.

There are four possible values, with the default being "None".

  • "None" or 0 - Does not return any Hilbert functions.
  • "One" or 1 - Returns the Hilbert function of an ideal which has the maximum possible sum of the total Betti numbers.
  • "AllMaxBettiSum" - Returns the Hilbert functions of all ideals that have the maximum possible sum of the total Betti numbers.
  • "All" - Returns the Hilbert functions of all ideals that have maximal total Betti numbers.

Further information

  • Default value: None
  • Function: maxBettiNumbers -- Upper bounds for total Betti numbers in a family of saturated ideals.
  • Option name: ResultsCount
